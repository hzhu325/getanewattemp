# 🔧 PartsPilot 汽配智能工作台

> 给做汽配生意的老爸减负：微信消息自动标记分级、17 位车架号（VIN）自动识别解码、
> 私聊自动回复、群聊生成回复草稿、询价自动建单、库存秒查——不用再一条条翻微信。

![Python](https://img.shields.io/badge/Python-3.10+-blue) ![Tests](https://img.shields.io/badge/tests-65%20passed-brightgreen) ![License](https://img.shields.io/badge/license-MIT-lightgrey)

## 它解决什么问题

| 以前 | 现在 |
|------|------|
| 微信群几百条消息，重要询价被淹没 | 消息自动打标（**询价 / VIN / 催单 / 售后 / 闲聊**）并按优先级标红，只看"待处理"清单 |
| 客户发车架号，要手动抄下来去查 | 自动从聊天内容里提取 17 位 VIN（容错空格、横线、O/0、I/1 混淆），本地校验 + 在线解码出品牌/车型/发动机型号 |
| "多少钱""有没有货"回到手软 | 私聊自动回复（可配话术）；**群聊只生成草稿，人点一下才发**，不乱说话 |
| 报价前翻库存本 | 询价自动匹配库存，草稿里直接附上"有货 E001 参考价 xx" |
| 半夜来消息 | 静默时段自动转草稿，第二天一并处理 |

**安全底线**：含价格/库存数量的内容**永远不会自动发出**，只进草稿由人确认。

## 三步上手（Windows 办公本即可，不吃显卡）

```
1. 双击「启动.bat」          ← 自动装依赖、起服务、打开浏览器
2. （可选）python scripts/seed_demo.py   ← 灌演示数据感受完整流程
3. 打开 http://127.0.0.1:8704 → 「模拟器」页发一句：
   要个大众迈腾2018年2.0T的发动机，多少钱
```

整个系统 = Python + SQLite + 原生网页，内存占用几十 MB，核显笔记本毫无压力。

## 怎么连上微信（两种方式，可并用）

微信侧使用官方 **小龙虾插件（ClawBot）**，本系统通过腾讯 `ilinkai.weixin.qq.com`
的 HTTP 接口收发消息——**程序跑在哪台电脑都行**，不需要动装微信的那台机器。

| 模式 | 适用 | 怎么开 |
|------|------|--------|
| **A. 直连**（推荐私聊） | 私聊自动回复 | 设 `CLAWBOT_ENABLED=1` 启动 → 设置页扫码绑定 |
| **B. Webhook 桥接** | 群消息、已有 OpenClaw 的场景 | 桥接端把消息 POST 过来，同步拿回复 |

详细步骤（含桥接示例代码）见 **[docs/CLAWBOT_SETUP.md](docs/CLAWBOT_SETUP.md)**。
不绑微信也能用：内置聊天模拟器可完整体验/测试全部逻辑。

## 功能页面

- **工作台**——今日数据、待处理会话（标红排序）、待发草稿一键发送、14 天消息量图
- **会话**——完整聊天记录、人工回复、按会话切换自动/草稿/关闭模式
- **询价单**——自动从聊天生成，缺什么信息一目了然（品牌/年份/VIN/型号），状态流转
- **库存**——发动机/变速箱/附件台账，自动参与询价匹配；**CSV 导入/导出**（下载模板，Excel 填好拖进来，几百条一次导入）
- **VIN 查询**——手动输入车架号解码（校验位验证 + 厂商/年款 + 可选 17vin 在线数据）
- **回复规则**——自定义"关键词 → 话术"，带预览测试
- **模拟器**——扮演客户发消息，右侧看系统的完整处理过程（打标/提取/决策）
- **设置**——店名、回复策略、静默时段、限流、**回复随机延迟（防机器痕迹）**、微信绑定、Webhook 令牌、**数据备份（每日自动 + 一键下载）**
- **会话页客户画像**——"历史询价 5 次 · 成交 2 次 · 上次成交：大众迈腾 EA888"，老客新客一眼分清

其他实用细节：

- **每日自动备份**：SQLite 在线备份保留最近 30 份（`data/backups/`），设置页可"备份并下载"存 U 盘
- **手机也能看**：界面是响应式的。把 `PARTSPILOT_HOST` 设为 `0.0.0.0`（同时设好密码），同一 WiFi 下手机浏览器访问 `http://电脑IP:8704` 即可

## 配置（都是可选的）

| 环境变量 | 作用 | 默认 |
|----------|------|------|
| `PARTSPILOT_PORT` | 端口 | 8704 |
| `PARTSPILOT_HOST` | 监听地址 | 127.0.0.1 |
| `PARTSPILOT_PASSWORD` | 管理后台密码（不设=免登录，仅限本机用） | 空 |
| `JISU_VIN_APPKEY` | [极速数据](https://www.jisuapi.com/api/vin/) VIN 解码（推荐：送100次，≈4.5分/次） | 空（用本地解码） |
| `TIANAPI_KEY` | [天行数据](https://www.tianapi.com/apiview/260) VIN 解码备选 | 空 |
| `SEVENTEEN_VIN_USER` / `SEVENTEEN_VIN_PASSWORD` | 17vin.com（需开户，见 [数据源对比](docs/VIN_PROVIDERS.md)） | 空 |
| `VIN_PROVIDER` | 强制指定数据源（jisuapi/tianapi/17vin） | 自动选择 |
| `CLAWBOT_ENABLED` | 开启微信小龙虾直连通道 | 0 |
| `CLAWBOT_BASE_URL` / `CLAWBOT_EP_*` | ilink 地址/端点覆盖 | 官方默认 |
