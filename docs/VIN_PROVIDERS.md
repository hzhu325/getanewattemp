# VIN 在线数据源选购指南（17vin 平替）

> 背景：17vin.com 需一次性开户约 ¥3000，对起步阶段太重。
> 调研结论（2026-07）：按次付费的 API 市场是更合适的选择，几块钱就能开始用。

## 对比（系统已内置全部三家的适配器）

| 数据源 | 免费额度 | 价格 | 门槛 | 返回字段 | 环境变量 |
|--------|---------|------|------|----------|----------|
| **极速数据 jisuapi**（推荐首选） | 注册送 **100 次** | ¥450/万次 ≈ **4.5分/次**；或包月 1000次/天 ¥300 | 注册即用，按量充值 | 品牌/厂商/车型/年款/排量/**发动机型号**/变速箱/相似车型列表 | `JISU_VIN_APPKEY` |
| **天行数据 tianapi** | 送 5 次 | ¥10 ≈ 150 次（≈6.5分/次） | 注册即用 | 品牌/车型/车系/年款/排量/**发动机型号**/变速箱类型/挡位数/厂商等 28 字段 | `TIANAPI_KEY` |
| 17vin.com | 无 | 一次性开户 ≈ ¥3000 | 高 | 最全（含 EPC/公告/进口车多源） | `SEVENTEEN_VIN_USER` + `SEVENTEEN_VIN_PASSWORD` |
| 本地离线解码（内置兜底） | 无限 | 免费 | 无 | 合法性校验/校验位/厂商(WMI)/年款 | 无需配置 |

**成本参考**：假设你爸每天收 20 个带 VIN 的询价，用极速数据一个月约 600 次 ≈ **¥27/月**。
先用送的 100 次试准确率，觉得行再充值，风险为零。

其他备选（未内置适配器，需要时可加）：[聚合数据 VIN码查询](https://www.juhe.cn/docs/api/id/283)、
[阿里云云市场多家](https://market.aliyun.com/apimarket/)（普遍也是 ¥450/万次档位）、
[极速数据精准版](https://www.jisuapi.com/api/vin2/)（¥0.5/次，号称精确到款型）。

## 接入步骤（以极速数据为例，5 分钟）

1. 注册 [jisuapi.com](https://www.jisuapi.com/api/vin/)，实名后申请「VIN车架号查询」→ 得到 `appkey`（送100次）
2. 编辑 `启动.bat`，在 `python run.py` 前加一行：
   ```bat
   set JISU_VIN_APPKEY=你的appkey
   ```
3. 重启系统 → 设置页应显示「已接入 极速数据」
4. VIN 查询页输入一个真实车架号验证；同时把设置里的「VIN 演示模式」关掉

天行数据同理：注册 [tianapi.com](https://www.tianapi.com/apiview/260) 申请「车架号VIN查询」，
设 `TIANAPI_KEY=你的key`。

## 多家并存与切换

- 同时配了多家时，系统自动选注册顺序里第一家已配置的（极速数据 → 天行 → 17vin）
- 用 `VIN_PROVIDER=tianapi` 可强制指定
- 任何在线源失败（欠费/超时/无结果）都自动降级为本地离线解码，**绝不会卡住自动回复**

## 验证建议

拿 3-5 个你们店里见过的真实 VIN（不同品牌：大众/丰田/国产各来一个），
在两家各查一遍对比发动机型号是否准确，再决定充值哪家。
两家都不准的车型（冷门进口车），流程上系统会自动提示客户拍铭牌照片兜底。
